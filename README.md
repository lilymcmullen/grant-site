# HDR Data Science Corps Grant Website

Site for NSF-funded Data Science Corps (DSC) program that was a collaboration between the University of Arizona (UA) and Lewis & Clark College (LC) to develop and test data science education grounded in biodiversity and conservation.

## About

A collaboration between:
- **University of Arizona** (Hispanic-Serving Institution, R1)
- **Lewis & Clark College** (Liberal Arts, Portland, OR)

## Building the Site

This is a Quarto website. To build locally:

```bash
quarto preview
```

To render:

```bash
quarto render
```

## Deployment

The site is deployed via GitHub Pages from the `/docs` folder.
